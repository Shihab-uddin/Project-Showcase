A lightweight and customizable portfolio showcase plugin to display your projects in a grid with live preview and snippet modes.

== Description ==

**Project Showcase by Gravth** allows you to display live web projects using a shortcode on any page or post. Each project supports a live preview URL, image snippet or a direct link to that site. Customize grid layout, colors, fonts, and button styles from the plugin settings panel.

**Features:**
- Create a custom portfolio post type
- Display projects in a responsive grid layout
- Choose between live preview, snippet view or direct live site visit link modes
- Fully customizable via settings (colors, layout, font sizes)
- Category filter support via shortcode attribute
- Clean and minimal design

Use the shortcode `[grvth_showcase]` or `[grvth_showcase category="your-category-slug"]` to embed the portfolio.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install directly via WordPress admin.
2. Activate the plugin through the 'Plugins' menu.
3. Go to "Project Showcase" > "Settings" to configure style and layout.
4. Add portfolio projects from the "Projects" menu.
5. Use the shortcode `[grvth_showcase]` on any page or post.

== Frequently Asked Questions ==

= Can I display specific categories? =  
Yes, use the `category` attribute: `[grvth_showcase category="web-design"]`

= Can I change the number of columns? =  
Yes, change it from the plugin settings.
