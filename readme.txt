== Description ==
Display and filter project showcases with live preview or code snippet view. Includes customizable layouts and settings.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory or install via the WordPress dashboard.
2. Activate the plugin.
3. Use the shortcode `[sp_portfolio]` to display the grid.
